## 🔧 Using Java Arguments for Better Performance

Java arguments tell your server how to use memory and CPU power. This guide explains which flags (small bits of text) to add and why.  

Common Java flags covered:

- `-Xms` and `-Xmx`: Control minimum and maximum memory (RAM) for the server.  
- `-XX:+UseG1GC`: Uses the Garbage Collector that improves performance.  
- `-XX:G1HeapRegionSize`: Helps with memory management, important for big servers.  
- `-XX:MaxGCPauseMillis`: Limits how long garbage collection can pause the server.  

We explain how much memory to allocate based on your hardware and player count.
